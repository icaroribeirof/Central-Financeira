# Central Financeira
Este é um projeto desenvolvido em PHP, CSS e JavaScript. O objetivo é possibilitar ao usuário um controle melhor de suas finanças, podendo lançar as despesas e receitas, cadastrar as categorias, seus cartões de crédito e ter uma visão analítica através dos dashboards.
